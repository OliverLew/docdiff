# 文档对比工具

自动对比两份纯文本文档，以不同颜色高亮形式输出。 增删部分分两行显示，无变化部分仅显示一行原文。

为方便修改，结果可复制粘贴至Word等软件中，颜色将会被保留。

在线demo： https://xlu.pythonanywhere.com/
